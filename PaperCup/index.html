Introducting PaperCup

PaperCup is a Windows video syncing software designed to avoid delay and lag normally found in other video syncing services. PaperCup accomplishes this by sending the state of one user's media player to the other's player.

For example, when User 1 pauses the video, PaperCup sends the changed state to User's 2 instance of the program. Then User 2's video pauses. User 2 can also do this same action to User 1.

PaperCup's networking functions over LAN, WLAN, and using port forwarding.
