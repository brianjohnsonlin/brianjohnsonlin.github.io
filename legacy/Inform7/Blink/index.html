A work of interactive fiction.

Blink was created with Inform and has IFID 4D0AF54F-41EB-47E6-95D8-288E82B96AF6. To play a work like this one, you need an interpreter program: many are available, among them Zoom for Mac OS X and for Unix; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
